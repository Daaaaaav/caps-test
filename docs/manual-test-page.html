🛡️ SQL Injection Testing Tool

Test Wazuh Security Monitoring Effectiveness

Authorization Required: Only test on systems you own or have explicit permission to test. Unauthorized testing is illegal.
ℹ️ How this works: This tool sends SQL injection payloads to your login endpoint. If Wazuh monitoring is working correctly, you should receive 403 Forbidden responses, and alerts should appear in your Wazuh dashboard and AI Security Reports page.

Test Results

📋 Verification Steps

  1. Check Laravel Logs: 
    type storage\logs\laravel.log | findstr /i "SQLI_DETECTED"
  2. Check Wazuh Dashboard: Navigate to Security Events and look for Level 12 alerts with "SQLI_DETECTED"
  3. Check AI Security Reports: Login as superadmin and verify the attack appears in the dashboard
